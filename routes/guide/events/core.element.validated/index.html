FormValidation • core.element.validated event

core.element.validated event

Triggered after validating a field element

The event object contains the following properties

  • element: HTMLElement — The field element which is being validated
  • elements: HTMLElement[] — The field elements
  • field: String — The field name
  • valid: Boolean — Can be true if the field element is valid, false otherwise.

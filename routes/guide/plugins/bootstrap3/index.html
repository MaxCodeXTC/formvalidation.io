Bootstrap 3 plugin

Integrate with Bootstrap 3 framework. Support the latest version of Bootstrap 3 (v3.3.7)

Use the Bootstrap plugin if your form is made in Bootstrap 4

Usage

The following piece of code is the starting point to validate the form made in Bootsrap 3:

...
document.addEventListener('DOMContentLoaded', function(e) { FormValidation.formValidation( document.getElementById('demoForm'), { fields: { ... }, plugins: { bootstrap3: new FormValidation.plugins.Bootstrap3(), ... }, } ); }); `} />

The sample code above assumes that the FormValidation files are placed inside the vendors directory. You might need to change the path depending on where you place them on the server.

Usage

The following piece of code is the starting point to validate the form made in Bulma:

...
document.addEventListener('DOMContentLoaded', function(e) { FormValidation.formValidation( document.getElementById('demoForm'), { fields: { ... }, plugins: { bulma: new FormValidation.plugins.Bulma(), ... }, } ); }); `} />

The sample code above assumes that the FormValidation files are placed inside the vendors directory. You might need to change the path depending on where you place them on the server.

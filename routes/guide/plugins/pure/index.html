Usage

The following piece of code is the starting point to validate the form made in Pure:

...
document.addEventListener('DOMContentLoaded', function(e) { FormValidation.formValidation( document.getElementById('demoForm'), { fields: { ... }, plugins: { pure: new FormValidation.plugins.Pure(), ... }, } ); }); `} />

The sample code above assumes that the FormValidation files are placed inside the vendors directory. You might need to change the path depending on where you place them on the server.
